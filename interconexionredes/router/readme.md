# ¿Qué hace un router?

Un **router** (enrutador) es un dispositivo de red que conecta diferentes redes entre sí y dirige el tráfico de datos entre ellas. Su función principal es recibir paquetes de datos de una red, determinar su destino y reenviarlos a la red adecuada, permitiendo la comunicación entre dispositivos de distintas redes (por ejemplo, entre una red local y el Internet).

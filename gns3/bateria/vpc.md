# Comandos para Configurar una Red en un VPC de GNS3


```bash
# Asignar dirección IP y máscara de subred
ip 192.168.1.10 255.255.255.0

# Configurar puerta de enlace predeterminada
ip gateway 192.168.1.1

# Configurar servidor DNS
ip dns 8.8.8.8

# Mostrar la configuración actual de red
show ip

# Probar conectividad con otro dispositivo
ping 192.168.1.1
```
